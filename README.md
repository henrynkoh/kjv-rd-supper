# 성찬식 커리큘럼 (Lord's Supper Curriculum)

Next.js app presenting key concepts from the Lord's Supper (성찬식) study—aligned with transcript content and supplementary research—in a curriculum format designed for solo learners.

## Features

- **13 categories** covering all major concepts
- **Step-by-step tables** with Concept, Transcript, Research, Action columns
- **Collapsible sections** for focused study
- **Quick navigation** to any category
- **No redundancy, no omissions** — comprehensive coverage

## Categories

1. 의식과 본질 (Rituals vs Essence)
2. 성찬식의 기원 (Lord's Supper Origins)
3. 유월절 어린 양 상징 (Passover Lamb Typology)
4. 인간의 본성 (Human Nature)
5. 상속의 조건 (Inheritance Conditions)
6. 결혼 비유 (Marriage Metaphor)
7. 죄와 속죄 (Sin & Atonement)
8. 부활 (Resurrection)
9. 성찬식 요소 (Communion Elements)
10. 합당한 참여 (Proper Partaking)
11. 이방인 시대와 의식 종료 (Gentile Age & Cessation)
12. 유월절 vs 대속죄일 (Passover vs Day of Atonement)
13. 요약 및 실행 (Summary & Action)

## Landing Page

The first page is a modern landing with:

- **Left sidebar** — scrollable nav to all 13 sub-sections; highlights active section (scroll spy)
- **Full content visible** — all curriculum tables displayed; emerald/teal/cyan gradient headers
- **Bottom-right** — clickable GitHub link (set `NEXT_PUBLIC_GITHUB_URL` in `.env` for your repo)
