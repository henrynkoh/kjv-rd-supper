# 빠른 시작 (Quick Start)

## 60초 안에 시작하기

```bash
cd kjv-rd-성찬식
npm install && npm run dev
```

브라우저에서 **http://localhost:3000** 접속.

---

## 3단계 요약

| # | 명령 | 설명 |
|---|------|------|
| 1 | `npm install` | 패키지 설치 |
| 2 | `npm run dev` | 개발 서버 실행 |
| 3 | 브라우저에서 `localhost:3000` | 커리큘럼 확인 |

---

## 프로덕션 배포

```bash
npm run build
npm start
```
