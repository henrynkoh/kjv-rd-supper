# 성찬식 커리큘럼 튜토리얼 (Step-by-Step Guide)

## 튜토리얼 개요

이 튜토리얼은 성찬식 커리큘럼을 **처음 접하는 학습자**를 위한 단계별 안내입니다. 약 30~45분 소요.

---

## Part 1: 환경 설정 (5분)

### Step 1.1 프로젝트 열기

터미널에서 프로젝트 폴더로 이동:

```bash
cd kjv-rd-성찬식
```

### Step 1.2 패키지 설치

```bash
npm install
```

### Step 1.3 개발 서버 실행

```bash
npm run dev
```

### Step 1.4 브라우저에서 열기

`http://localhost:3000` 접속

---

## Part 2: 첫 번째 카테고리 학습 (10분)

### Step 2.1 1번 카테고리 열기

- **1. 의식과 본질 (Rituals vs Essence)** 제목 클릭
- 섹션이 펼쳐지며 테이블이 보입니다.

### Step 2.2 테이블 읽기

| 읽는 순서 | 열(Column) | 하는 일 |
|-----------|------------|---------|
| 1 | Concept | 영어로 된 핵심 개념 파악 |
| 2 | Transcript (원문) | 강의 원문 요약으로 맥락 이해 |
| 3 | Research (보조) | 성경/연구 자료로 보완 이해 |
| 4 | Action | 제시된 과제 수행 |

### Step 2.3 Action 수행 예시

예: "Reflect: Do I seek ritual for stability or for the substance?"

→ 자신의 마음에서 의식을 원하는 이유를 생각해 봅니다.

---

## Part 3: 전체 커리큘럼 탐색 (15분)

### Step 3.1 Quick Navigation 사용

상단의 pill 버튼으로 원하는 카테고리로 바로 이동:

- **1. 의식과 본질** 클릭 → 1번 섹션으로 스크롤
- **9. 성찬식 요소** 클릭 → 9번 섹션으로 스크롤

### Step 3.2 Expand all / Collapse all

- **Expand all**: 13개 카테고리 모두 펼치기 (한눈에 보기)
- **Collapse all**: 모두 접기 (탐색용)

### Step 3.3 권장 학습 순서

1. 1번 → 2번 → 3번 … 13번 순서대로 진행
2. 각 카테고리 내 Step 1, 2, 3… 순서대로 읽기
3. Action은 기록하거나 말로 정리해 보기

---

## Part 4: 핵심 개념 복습 체크리스트 (5분)

다음 질문에 답할 수 있으면 튜토리얼을 마친 것입니다.

1. 성찬식은 **예수님의 몸**에 관한 이야기라는 점을 설명할 수 있나요?
2. **빵**과 **포도주**가 각각 무엇을 상징하는지 말할 수 있나요?
3. **유월절 어린 양**과 예수님의 연결을 한 문장으로 요약할 수 있나요?
4. **이방인 시대**와 의식(침례, 성찬식) 종료의 관계를 이해했나요?
5. 성찬식의 **본질**이 의식 자체가 아니라 **믿음**이라는 점을 이해했나요?

---

## Part 5: 심화 학습 (선택)

- 한 카테고리를 선정해 성경 구절 직접 찾아보기
- `curriculum-data.ts`에서 Transcript/Research 원문을 더 긴 버전으로 확장하기
- 학습 노트를 별도 문서로 정리하기

---
